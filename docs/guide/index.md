## 目录

- DOCS

可以理解成 VuePress 项目的 src 文件夹

> docs/.vuepress/config.ts：配置文件的入口文件  
> docs/.vuepress/dist：项目打包生成的文件夹  
> docs/public：存放静态资源文件夹(用于存放Logo等资源)  
> docs/guide：存放项目指南介绍的文件夹  
> docs/note：存放笔记文档的文件夹  

- NODE_MODULES

安装 node 后用来存放用包管理工具下载安装的包的文件夹
